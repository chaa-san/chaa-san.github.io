This Is How I Built This Page Using FLASK

Installation

To install Flask:

  • Check if Python is installed: python --version
  • Set-up a virtual environment: python3 -m venv venv.
  • Activaye the virtual environment: .\venv\Scripts\activate
  • Install Flask: pip install flask.

Building the Page

To create the webpage, I used HTML and CSS for the structure and styling. The Flask application works as the backend to handle requests and display the webpage. Such as:

  • Using the Flask app's route that loads and displays the HTML file.

Difficulties Encountered

- Issue: Setting up Flask on Windows required enabling Python in the system PATH.
- Solution: I went into my system settings and added Python's installation folder to the PATH variable, so the terminal could recognize Python commands.

- Issue: Making the webpage accessible online.
- Solution: I used GitHub Pages to host the static content and ensure the project was accessible for testing and submission.
